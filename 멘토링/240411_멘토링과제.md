# JDK vs JRE
## JDK
★자바개발도구(Java Development Kit)
★JDK = JRE + 개발에 필요한 실행파일 (javac.exe 등)

## JRE
★자바실행환경(Java Runtime Environment)
- 자바로 작성된 응용프로그램이 실행되기 위한 최소환경
★JRE = JVM + 클래스라이브러리(Java API)
